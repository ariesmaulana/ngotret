# Zen Note - Vue Crepe

A Vue.js application using Milkdown's Crepe editor for markdown editing.

## Project Structure

This project is based on the [Milkdown vue-crepe example](https://github.com/Milkdown/examples/tree/main/vue-crepe).

## Features

- **Milkdown Crepe Editor**: A modern WYSIWYG markdown editor
- **Vue 3**: Built with Vue 3 and Composition API
- **TypeScript**: Full TypeScript support
- **Vite**: Fast development server and build tool

## Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

The application will be available at `http://localhost:5173/`
